# Project Overview

This project focuses on creating a professional starter CV tailored for a Software Development role. The goal is to design a clear, concise, and well-structured resume that highlights my technical skills, education, and professional experience relevant to the tech industry.

The CV emphasises my transition into software development, particularly focusing on front-end development technologies such as JavaScript and React.js.

This resume will continue to evolve as I gain more experience and complete additional coding projects throughout the course.

# Project Objectives

The objectives of this project are:

- To create a professional CV suitable for software development roles.

- To highlight relevant technical skills, education, and transferable experience.

- To present information in a clear and professional format that attracts hiring managers.

- To build a foundational CV that can be updated as new projects and skills are developed.
